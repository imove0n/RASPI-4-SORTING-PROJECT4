BLYNK SETUP CHECKLIST - Follow in Order
========================================

PHONE (2 min)
□ 1. Install "Blynk IoT" app
□ 2. Create account with email + password
□ 3. Verify email

COMPUTER - https://blynk.cloud (5 min)
□ 4. Login with same email/password
□ 5. Templates → + New Template
     Name: DHT22 Sensor
     Hardware: Raspberry Pi
     Connection: WiFi
□ 6. Datastreams → + New Datastream → Virtual Pin
     V0: Temperature (Double, 0-50, default 25)
□ 7. Datastreams → + New Datastream → Virtual Pin
     V1: Humidity (Double, 0-100, default 50)
□ 8. Click Save
□ 9. Devices → + New Device → From Template → DHT22 Sensor
□ 10. **COPY AUTH TOKEN** (the long string of letters/numbers)

PHONE APP
□ 11. Open app → Tap "My DHT22" device
□ 12. Tap wrench icon → Tap +
□ 13. Add Gauge widget → Set to Temperature (V0)
□ 14. Tap + again
□ 15. Add Gauge widget → Set to Humidity (V1)
□ 16. Tap Save

RASPBERRY PI
□ 17. Open: dht22_blynk.py
□ 18. Replace "YourAuthTokenHere" with your token
□ 19. Save file
□ 20. Run: python3 dht22_blynk.py

VERIFY
□ 21. See "Connected to Blynk!" in terminal
□ 22. See data being sent every 5 seconds
□ 23. Check phone - gauges should show live data!

✓ DONE! Your phone shows live sensor data!
