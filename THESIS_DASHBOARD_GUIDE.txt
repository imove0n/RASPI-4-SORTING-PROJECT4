HOW TO START THE DASHBOARD:
============================

1. Open Terminal

2. Navigate to project folder:
   cd "Desktop/RASPI 4 SORTING PROJECT"

3. Run the dashboard server:
   python3 dht22_web_dashboard.py

4. You'll see output like:
   ============================================================
   Server starting on: http://192.168.100.18:5000
   ============================================================

5. Open that URL in ANY browser:
   - On Raspberry Pi: http://localhost:5000
   - On your phone: http://192.168.100.18:5000
   - On laptop: http://192.168.100.18:5000

   (Your IP address may be different!)

TROUBLESHOOTING:
================

Issue: "Cannot connect to sensor"
Fix: Check DHT22 wiring on GPIO4

Issue: Dashboard doesn't load
Fix: Make sure server is running (python3 dht22_web_dashboard.py)

Issue: Phone can't access dashboard
Fix: Ensure phone on same WiFi as Raspberry Pi

Issue: Numbers not updating
Fix: Refresh browser (F5 or Ctrl+R)

Issue: Counts show 0
Fix: This is normal on first load, wait 2 seconds for update
