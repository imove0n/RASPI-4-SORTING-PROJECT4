BLYNK SETUP - COMPLETE BEGINNER'S GUIDE
========================================

PART 1: PHONE APP SETUP (2 minutes)
====================================

Step 1: Install the App
------------------------
On your phone:
□ Open Google Play Store (Android) or App Store (iOS)
□ Search for: "Blynk IoT"
□ Look for app by "Blynk Inc." (has green/blue icon)
□ Tap "Install" or "Get"
□ Wait for download to complete
□ Tap "Open"


Step 2: Create Your Account
----------------------------
In the Blynk app:
□ You'll see a welcome screen
□ Tap "Sign Up" or "Create New Account"
□ Enter your email address
□ Create a password (write it down!)
□ Tap "Sign Up" or "Create Account"
□ Check your email for verification code
□ Enter the verification code
□ You're in! (Don't close the app yet)


PART 2: COMPUTER SETUP (5 minutes)
===================================

Step 3: Login on Computer
--------------------------
On your computer browser:
□ Go to: https://blynk.cloud
□ Click "Log In"
□ Enter the SAME email and password you used on phone
□ Click "Log In"
□ You should see the Blynk dashboard


Step 4: Create a Template
--------------------------
What you'll see: Left side menu with options

□ Click "Templates" (in left menu)
□ Click the blue "+ New Template" button (top right)

A popup appears asking for details:
□ Template Name: Type "DHT22 Sensor" (exactly like this)
□ Hardware: Click dropdown, select "Raspberry Pi"
□ Connection Type: Click dropdown, select "WiFi"
□ Click "Done" button

You should now see your template page!


Step 5: Add Datastreams (This tells Blynk what data to expect)
---------------------------------------------------------------
What you'll see: Template page with tabs at top

□ Click the "Datastreams" tab (near top of page)
□ Click "+ New Datastream" button (blue button)
□ Select "Virtual Pin" from dropdown

FIRST DATASTREAM (Temperature):
A form appears:
□ Virtual Pin: Type "V0" (capital V, then zero)
□ Name: Type "Temperature"
□ Data Type: Select "Double" from dropdown
□ Units: Leave blank or type "°C"
□ Min Value: Type "0"
□ Max Value: Type "50"
□ Default Value: Type "25"
□ Click "Create" button

SECOND DATASTREAM (Humidity):
□ Click "+ New Datastream" again
□ Select "Virtual Pin"

Fill in the form:
□ Virtual Pin: Type "V1" (capital V, then number one)
□ Name: Type "Humidity"
□ Data Type: Select "Double" from dropdown
□ Units: Type "%"
□ Min Value: Type "0"
□ Max Value: Type "100"
□ Default Value: Type "50"
□ Click "Create" button

□ Click "Save" button (top right corner of page)

You should see 2 datastreams listed: V0 Temperature and V1 Humidity


Step 6: Create Web Dashboard (Optional but cool to see on computer)
--------------------------------------------------------------------
□ Click "Web Dashboard" tab (at top)
□ You'll see a blank canvas

ADD TEMPERATURE GAUGE:
□ Look at left side - you'll see widgets
□ Find "Gauge" widget (looks like a speedometer)
□ Drag it onto the canvas (click and hold, then drag)
□ A settings panel appears on right
□ Datastream: Select "Temperature (V0)" from dropdown
□ Title: Type "Temperature"
□ Click anywhere outside to close settings

ADD HUMIDITY GAUGE:
□ Drag another "Gauge" widget onto canvas
□ Settings panel appears
□ Datastream: Select "Humidity (V1)"
□ Title: Type "Humidity"
□ Click outside to close

□ Click "Save" button (top right)


Step 7: Create Your Device (IMPORTANT - This gets you the token!)
------------------------------------------------------------------
□ Click "Search" (magnifying glass icon, top left)
□ Click "Devices"
   OR click "Devices" in left menu

□ Click "+ New Device" button (blue, top right)
□ Select "From template"

A list appears:
□ Click on "DHT22 Sensor" (the template you created)
□ Device Name: Type "My DHT22" (or any name you like)
□ Click "Create" button

*** CRITICAL STEP - GET YOUR AUTH TOKEN ***

You'll see device information page with lots of details

□ Look for "BLYNK_AUTH_TOKEN" or "Auth Token"
□ You'll see a long string like: "abc123XYZ789..."

COPY THIS TOKEN! You have 3 options:

Option 1 (Easiest):
  □ Click the "Copy" icon next to the token
  □ Token is now copied to clipboard

Option 2 (Manual):
  □ Click and drag to select the entire token
  □ Press Ctrl+C (Windows/Linux) or Cmd+C (Mac)

Option 3 (Write it down):
  □ CAREFULLY write down the ENTIRE token on paper
  □ Make sure every letter and number is correct!

SAVE THIS TOKEN! You'll need it in Step 9!


PART 3: PHONE APP DASHBOARD (2 minutes)
========================================

Step 8: Setup Phone Dashboard
------------------------------
Go back to Blynk app on your phone:

□ You should see your device "My DHT22" in the list
□ If not, pull down to refresh

□ Tap on "My DHT22" device
□ You'll see a blank screen
□ Tap the wrench/settings icon (top right corner)

ADD TEMPERATURE WIDGET:
□ Tap the "+" button (usually bottom right)
□ Scroll and find "Gauge" widget
□ Tap on "Gauge"
□ The widget appears on screen
□ Tap on the gauge widget to configure it

Settings screen appears:
□ Datastream: Tap it, select "Temperature (V0)"
□ Label: Type "Temperature"
□ Look at the gauge preview - it should show 0-50 range
□ Tap "Save" or back arrow

ADD HUMIDITY WIDGET:
□ Tap "+" again
□ Select "Gauge" widget
□ Tap on the new gauge to configure

Settings:
□ Datastream: Select "Humidity (V1)"
□ Label: Type "Humidity"
□ Tap "Save" or back arrow

□ Tap "Save" button (top right) to finish dashboard

Your phone is ready! The gauges might show "0" or "N/A" - this is normal until we connect.


PART 4: CONFIGURE RASPBERRY PI SCRIPT (1 minute)
=================================================

Step 9: Add Your Auth Token to Python Script
---------------------------------------------
On your Raspberry Pi:

□ Open file manager OR use terminal
□ Navigate to: Desktop/RASPI 4 SORTING PROJECT/
□ Find file: dht22_blynk.py
□ Right-click → Open with → Text Editor
   OR in terminal: nano dht22_blynk.py

You'll see the code. Look for this line (near the top):
  BLYNK_AUTH = "YourAuthTokenHere"

□ Delete "YourAuthTokenHere" (keep the quotes "")
□ Paste your Auth Token between the quotes

BEFORE:
  BLYNK_AUTH = "YourAuthTokenHere"

AFTER (example):
  BLYNK_AUTH = "abc123XYZ789def456GHI-jkl"

Make sure:
  - Token is between the quotes ""
  - No spaces before or after token
  - All letters and numbers copied correctly

□ Save the file:
  - Text Editor: Click "Save" or Ctrl+S
  - Nano: Press Ctrl+O, Enter, then Ctrl+X

DONE! Script is configured!


PART 5: TEST IT! (30 seconds)
==============================

Step 10: Run the Script
------------------------
On Raspberry Pi, open terminal:

□ Type: cd "Desktop/RASPI 4 SORTING PROJECT"
□ Press Enter
□ Type: python3 dht22_blynk.py
□ Press Enter

What you should see:
  Initializing DHT22 sensor...
  Connecting to Blynk...
  ✓ Connected to Blynk!
  ============================================================
  DHT22 → BLYNK - Monitoring Started
  ============================================================
  [Time] Data #1 sent to Blynk:
    Temperature: 30.5°C
    Humidity: 68.2%

If you see this - SUCCESS! 🎉


Step 11: Check Your Phone
--------------------------
□ Open Blynk app on your phone
□ Make sure you're connected to internet
□ Tap on "My DHT22" device
□ Watch the gauges!

Within 5-10 seconds you should see:
  - Temperature gauge showing live temperature
  - Humidity gauge showing live humidity
  - Values updating every 5 seconds

SUCCESS! You're monitoring your sensor from your phone! 📱


TROUBLESHOOTING
===============

Problem: "Could not connect to Blynk"
-------------------------------------
Check:
□ Raspberry Pi is connected to WiFi (try: ping google.com)
□ Auth Token is EXACTLY the same (no extra spaces)
□ Token is between the quotes ""
□ You created the device on blynk.cloud


Problem: Phone shows "0" or "Offline"
--------------------------------------
Check:
□ Python script is running on Raspberry Pi
□ You see "Connected to Blynk!" in terminal
□ Wait 10 seconds for first data
□ Pull down on phone app to refresh


Problem: "Sensor read error"
-----------------------------
Check:
□ DHT22 wires are connected firmly
□ Run: python3 test_gpio.py (to test sensor)
□ Wait 15 seconds, errors are normal occasionally


Problem: Can't find Auth Token
-------------------------------
□ Go to: https://blynk.cloud
□ Click "Devices" (left menu)
□ Click on "My DHT22"
□ Look for "Device Info" section
□ Auth Token is there - click copy icon


NEED MORE HELP?
===============
□ Make sure you followed every step with checkboxes □
□ Check you completed all □ checkmarks
□ Try stopping script (Ctrl+C) and running again
□ Ask for help and mention which step number you're stuck on


STOPPING THE SCRIPT
===================
□ Press Ctrl+C in terminal
□ Script will show statistics and exit cleanly
□ To run again: python3 dht22_blynk.py


YOU'RE DONE! 🎉
===============
Your DHT22 sensor is now connected to your phone via Blynk!
You can view temperature and humidity from anywhere with internet.
