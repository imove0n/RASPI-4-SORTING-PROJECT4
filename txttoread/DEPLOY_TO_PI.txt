STEP 1: Copy Model to Raspberry Pi
-----------------------------------

METHOD A: USB Drive
1. Copy model.pt to USB drive
2. Plug USB into Raspberry Pi
3. Copy to project folder:
   cp /media/usb/model.pt ~/Desktop/RASPI\ 4\ SORTING\ PROJECT/

METHOD B: Network Transfer (if on same WiFi)
1. On Pi, get IP address: hostname -I
2. On your PC, use WinSCP or similar
3. Connect to Pi
4. Copy model.pt to: ~/Desktop/RASPI 4 SORTING PROJECT/

METHOD C: Google Drive
1. Upload model.pt to Google Drive
2. On Pi, download from Google Drive
3. Move to project folder


STEP 2: Install YOLO on Raspberry Pi
-------------------------------------

Open terminal on Pi:

cd ~/Desktop/RASPI\ 4\ SORTING\ PROJECT/

Install Ultralytics YOLO:

pip3 install ultralytics

This takes 5-10 minutes!

Wait for "Successfully installed ultralytics"

TROUBLESHOOTING:
================

Issue: "No module named ultralytics"
Fix: pip3 install ultralytics

Issue: Model file not found
Fix: Check path to model.pt
     Make sure it's in same folder as script

Issue: Camera not working
Fix: Test with existing script first:
     python3 camera_test.py

Issue: Detection too slow
Fix: Normal on Pi! 5-10 FPS is expected
     Can use smaller model (yolov8n) for speed

Issue: Low accuracy
Fix: Need more training images
     40 images = 80-85% accuracy
     100 images = 90-95% accuracy

Issue: Not detecting tomatoes
Fix: Check confidence threshold in script
     Lower threshold = more detections
