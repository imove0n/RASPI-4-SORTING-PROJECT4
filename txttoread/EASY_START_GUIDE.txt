========================================================
SUPER EASY START GUIDE - FOR BEGINNERS
========================================================

I KNOW YOU'RE OVERWHELMED! Let's make this SIMPLE!


WHAT YOU HAVE:
==============
✓ Raspberry Pi 4
✓ DHT22 temperature sensor (working)
✓ Pi Camera v2 (working)
✓ 6 tomatoes to test with
✓ Beautiful web dashboard


STEP-BY-STEP - DO THIS NOW:
============================

STEP 1: Open Terminal
----------------------
Click the black terminal icon at top of screen


STEP 2: Go to Project Folder
-----------------------------
Type this and press ENTER:

cd "Desktop/RASPI 4 SORTING PROJECT"


STEP 3: Run the Camera Classifier
----------------------------------
Type this and press ENTER:

python3 tomato_classifier_with_preview.py


STEP 4: What Happens Next
--------------------------
A WINDOW will pop up showing what camera sees!
(Like a video preview)

You'll see:
- Live camera view
- Text saying "Press SPACE to classify tomato"
- Counters at bottom (Unripe: 0, Ripe: 0, Overripe: 0)


STEP 5: Classify Your First Tomato
-----------------------------------
1. Hold one tomato in front of camera
2. Make sure tomato fills most of the view
3. Press SPACE BAR on keyboard
4. Wait 1 second
5. You'll see result: "RIPE" or "UNRIPE" or "OVERRIPE"
6. Image automatically saves!


STEP 6: Try All 6 Tomatoes
---------------------------
Repeat for each tomato:
- Hold tomato in front of camera
- Press SPACE
- See classification
- Next tomato!


STEP 7: When Done
-----------------
Press 'q' key to quit
You'll see final summary of all classifications


WHERE ARE MY IMAGES?
=====================

All classified images are saved to:

Desktop/RASPI 4 SORTING PROJECT/classified_tomatoes/

To view them:
1. Open file manager
2. Go to Desktop
3. Go to "RASPI 4 SORTING PROJECT" folder
4. Open "classified_tomatoes" folder
5. Double-click images to view!

Each image has:
- The classification written on it (RIPE, UNRIPE, etc)
- Filename shows category and time


TROUBLESHOOTING:
================

Problem: No window appears
Solution: Look behind other windows, it might be hidden
          Or try: Alt+Tab to switch windows

Problem: Window is black
Solution: Camera might be covered
          Check camera lens is not blocked

Problem: Can't see tomato clearly
Solution: Move tomato closer to camera
          Adjust lighting (turn on room light)

Problem: Wrong classification
Solution: Normal! Simple color detection is 75-85% accurate
          YOLO will be better (we'll do that later)

Problem: Terminal says "error"
Solution: Make sure camera ribbon cable is connected
          Blue side faces USB ports


WHAT EACH FILE DOES:
====================

UNDERSTANDING THE RESULTS:
===========================

UNRIPE (Green color detected)
- Tomato is not ready to eat
- Needs more time to ripen
- Store at room temperature

RIPE (Red/orange color detected)
- Tomato is perfect for eating
- Sell or eat immediately
- Best flavor

OVERRIPE (Dark red detected)
- Very soft, may be too ripe
- Use for cooking/sauce
- Don't store long
