STEP-BY-STEP INSTRUCTIONS:
===========================

STEP 1: Create Roboflow Account
--------------------------------
1. Go to: https://roboflow.com
2. Click "Sign Up" (top right)
3. Sign up with Google account (fastest)
4. Verify email if needed
5. You'll see the Roboflow dashboard

✓ Account created!


STEP 2: Create New Project
---------------------------
1. Click "Create New Project"
2. Project Name: "Tomato Detector"
3. Project Type: Select "Object Detection"
4. What will you detect?: "Tomatoes"
5. Click "Create Project"

✓ Project created!


STEP 3: Upload Your Images
---------------------------
1. Click "Upload Data" or drag and drop
2. Select all 40 images from your folder
3. Wait for upload to complete
4. Click "Finish Uploading"
5. Click "Save and Continue"

✓ Images uploaded!


STEP 4: Annotate (Label) Images
--------------------------------
This is where you draw boxes!

For EACH image:

1. You'll see an image with tools on the left
2. Select "Bounding Box" tool (rectangle icon)
3. Click and drag around the tomato
   - Start at top-left corner of tomato
   - Drag to bottom-right corner
   - Box should tightly fit the tomato
4. A label box appears
5. Type the class name:
   - If ripe (red/orange): Type "ripe"
   - If unripe (green): Type "unripe"
6. Press Enter
7. If there are multiple tomatoes in image:
   - Draw another box
   - Label it
   - Repeat for all tomatoes
8. Click "Save" or press Ctrl+S
9. Click "Next Image" (or press 'n')

Repeat for all 40 images!


KEYBOARD SHORTCUTS (Faster!):
------------------------------
- n = Next image
- p = Previous image
- Ctrl+S = Save current image
- Delete = Remove selected box
- Esc = Cancel current box


LABELING TIPS:
==============

Good Box:
---------
┌──────────┐
│   🍅     │  ← Tight around tomato
└──────────┘  ← Small border

Bad Box (Too Loose):
--------------------
┌──────────────┐
│      🍅      │  ← Too much space
└──────────────┘

Perfect!
--------
- Include whole tomato
- Small margin around edges
- Don't cut off any part


CLASS NAMES (Use exactly these):
=================================
ripe     → For red/orange tomatoes
unripe   → For green tomatoes

(Just 2 classes since you don't have rotten!)


STEP 5: Generate Dataset
-------------------------
After labeling all images:

1. Click "Generate" (top right)
2. Or go to "Generate" tab
3. Preprocessing: Leave default
4. Augmentation: Select these (optional but recommended):
   ☑ Flip: Horizontal
   ☑ Rotation: ±15°
   ☑ Brightness: ±15%
   ☑ Blur: Up to 1px
5. This creates more training variations!
6. Number of images: Will show ~80-120 (with augmentations)
7. Click "Generate"
8. Wait 1-2 minutes

✓ Dataset generated!


STEP 6: Export for YOLO
------------------------
1. Click "Export" or "Download"
2. Format: Select "YOLOv8"
3. Click "Download ZIP"
4. Save to your computer
5. Extract the ZIP file

You'll get a folder with:
- train/
  - images/
  - labels/
- valid/
  - images/
  - labels/
- data.yaml

✓ Dataset ready for training!

TROUBLESHOOTING:
================

Issue: Can't draw box
Fix: Click bounding box tool first (rectangle icon)

Issue: Wrong label
Fix: Click the box → Edit label → Change name

Issue: Deleted box by accident
Fix: Press Ctrl+Z to undo

Issue: Upload failed
Fix: Check image file format (JPG/PNG only)
      Check file sizes (<10MB each)
