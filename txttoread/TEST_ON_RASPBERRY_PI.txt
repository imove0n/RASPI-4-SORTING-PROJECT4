TROUBLESHOOTING:
================

Error: "No module named ultralytics"
Fix:
pip3 install ultralytics


Error: "Could not find model best.pt"
Fix:
ls best.pt
(Check file is there)
If not, copy it again from PC


Error: "No module named picamera2"
Fix:
sudo apt update
sudo apt install -y python3-picamera2


Error: "Camera not detected"
Fix:
1. Check camera ribbon cable
2. Enable camera: sudo raspi-config
3. Test with: python3 camera_test.py


Error: Detection very slow (1-2 FPS)
Fix:
This is normal on Pi! 5-10 FPS is expected.
Model is optimized but Pi has limited power.


Error: Low accuracy / wrong detections
Fix:
Normal with 40 training images (80-85% accuracy)
To improve:
- Take 40-60 more photos
- Label them
- Retrain model with 80-100 total images
