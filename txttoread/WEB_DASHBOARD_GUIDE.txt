HOW TO START THE WEB DASHBOARD:
================================

Step 1: Open Terminal on Raspberry Pi
--------------------------------------
cd "Desktop/RASPI 4 SORTING PROJECT"
python3 dht22_web_dashboard.py


Step 2: Look for the IP Address
--------------------------------
The server will show you something like:

  Server starting on: http://192.168.100.18:5000

  Access from:
    - This device:  http://localhost:5000
    - Your phone:   http://192.168.100.18:5000  ← USE THIS!
    - Any browser:  http://192.168.100.18:5000

Copy the IP address (yours will be different!)


Step 3: Open on Your Phone
---------------------------
1. Make sure your phone is on the SAME WiFi as your Raspberry Pi
2. Open your phone's web browser (Chrome, Safari, etc.)
3. Type in the URL: http://192.168.100.18:5000
   (Replace with YOUR IP address from Step 2)
4. Press Enter/Go

TROUBLESHOOTING:
================

Problem: "Can't connect" or "Site can't be reached"
----------------------------------------------------
Solution:
  1. Make sure Raspberry Pi and phone are on SAME WiFi
  2. Check the script is running (should see the server messages)
  3. Try the IP address in your phone's browser again
  4. Make sure you include :5000 at the end


Problem: Dashboard shows "Offline"
-----------------------------------
Solution:
  1. Check DHT22 wires are connected
  2. Script might be starting up (wait 5 seconds)
  3. Dashboard will auto-reconnect


Problem: "Address already in use"
----------------------------------
Solution:
  Another instance is running. Stop it first:
    pkill -f dht22_web_dashboard.py
  Then start again


Problem: Need to find IP address again
---------------------------------------
Solution:
  Run this command:
    hostname -I
  The first number is your IP address


STOPPING THE SERVER:
====================

Press Ctrl+C in the terminal where the server is running.


RUNNING IN BACKGROUND (Optional):
==================================

To run the server in background so you can close terminal:

  nohup python3 dht22_web_dashboard.py > dashboard.log 2>&1 &

To stop background server:
  pkill -f dht22_web_dashboard.py
