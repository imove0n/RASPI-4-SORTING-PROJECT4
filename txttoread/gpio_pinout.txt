Raspberry Pi 4 GPIO Pinout for DHT22 Sensor
=============================================

Looking at your Raspberry Pi with the USB ports facing you:

LEFT SIDE (ODD numbers)          RIGHT SIDE (EVEN numbers)
┌─────────────────┐             ┌─────────────────┐
│  3.3V      [1]  │             │  [2]  5V        │  ← Power options
│  GPIO2     [3]  │             │  [4]  5V        │
│  GPIO3     [5]  │             │  [6]  GND       │  ← Ground
│ →GPIO4     [7]  │ ← DATA PIN  │  [8]  GPIO14    │
│  GND       [9]  │             │ [10]  GPIO15    │
│  GPIO17   [11]  │             │ [12]  GPIO18    │
│  GPIO27   [13]  │             │ [14]  GND       │
│  GPIO22   [15]  │             │ [16]  GPIO23    │
│  3.3V     [17]  │             │ [18]  GPIO24    │
│  GPIO10   [19]  │             │ [20]  GND       │
│  GPIO9    [21]  │             │ [22]  GPIO25    │
│  GPIO11   [23]  │             │ [24]  GPIO8     │
│  GND      [25]  │             │ [26]  GPIO7     │
└─────────────────┘             └─────────────────┘

DHT22 WIRING (using female-to-female jumpers):
==============================================

DHT22 has 3 or 4 pins (left to right when facing the sensor):

Option 1: Using 3.3V power (RECOMMENDED)
-----------------------------------------
DHT22 Pin 1 (VCC/+)  →  Pi Pin 1  (3.3V)      [Red wire typically]
DHT22 Pin 2 (DATA)   →  Pi Pin 7  (GPIO4)     [Yellow/White wire]
DHT22 Pin 3 (NC)     →  Not connected
DHT22 Pin 4 (GND/-)  →  Pi Pin 6  (GND)       [Black/Brown wire]

Option 2: Using 5V power (alternative)
---------------------------------------
DHT22 Pin 1 (VCC/+)  →  Pi Pin 2  (5V)        [Red wire]
DHT22 Pin 2 (DATA)   →  Pi Pin 7  (GPIO4)     [Yellow/White wire]
DHT22 Pin 3 (NC)     →  Not connected
DHT22 Pin 4 (GND/-)  →  Pi Pin 6  (GND)       [Black/Brown wire]

SIMPLE 3-WIRE CONNECTION:
=========================
1. Red wire:    DHT22 + pin  →  Pi Pin 1 (3.3V) or Pin 2 (5V)
2. Yellow wire: DHT22 DATA   →  Pi Pin 7 (GPIO4)
3. Black wire:  DHT22 - pin  →  Pi Pin 6 (GND)

NOTE: Some DHT22 modules come with built-in resistor and only have 3 pins.
If yours has 4 pins, pin 3 is usually not connected (NC).
